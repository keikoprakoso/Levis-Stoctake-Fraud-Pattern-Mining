# LEVIS Stocktake Analysis & Fraud Pattern Mining

## 🚀 Quick Start

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Run Complete Analysis
```bash
python run_analysis.py
```

### 3. Launch Interactive Dashboard
```bash
python run_dashboard.py
```
